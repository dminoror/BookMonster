# BookMonster </br>
</br>
這是一個專為看本所設計的程式，是針對圖檔太大或奇怪格式導致載入較慢的一種浪費記憶體的解決方案。</br>

## 特色說明 </br>
 * 打開任意圖片後，程式會將該圖片同一資料夾下的其餘圖片預載入記憶體，此情況下翻頁幾乎沒有任何延遲。</br>
 * 記憶體的使用上限預設為總記憶體的一半，可以在設定中更改。</br>

## 功能說明 </br>
 * 將圖片，或含有圖片的資料夾拖入視窗內即可開始載入，或可以直接設為預設程式。</br>
 * 方向鍵 左/右 翻頁。圖片的排列順序以名稱為準；當達到盡頭時左右會有漸層提示。</br>
 * 方向鍵 上/下 切換本子資料夾。資料夾排序以名稱為準，資料夾內不含任何圖片則不列入計算；當達到盡頭時上下會有漸層提示。</br>
 * 右鍵選單可切換至卷軸模式。在此模式下左右翻頁無效，滑鼠滾動速度在設定中可調整。</br>
 * "其他看圖軟體" 設定路徑後可以快速啟動。這可能在需要本程式不支援的功能時(例如旋轉)十分有用。開啟其他軟體後本程式會停止預載圖片。</br>
 * 以上熱鍵在設定中都可以修改。</br>
